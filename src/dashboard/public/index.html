เปิด Spreadsheet

📌 คำสั่งบน Telegram (Commands)

  • 📷 ส่งรูปสลิป — วิเคราะห์ + บันทึกลง Sheet อัตโนมัติ
  • /report — สรุปยอดวันนี้ (ทุกบัญชี)
  • /report 4321 — เฉพาะบัญชีลงท้าย 4321
  • /report all 2026-06-01 — เลือกวัน/ทั้งหมด
  • /add 4321 500 โอน — บันทึกเองไม่มีสลิป
  • /undo 4321 — ลบรายการล่าสุดของบัญชีนั้น
  • /clear 4321 · /clear_all — ล้างยอดวันนี้
  • /wipe 4321 · /wipe_all — ล้างทั้งหมด + ลบรูป Drive
  • /status — สถานะบอท + token
  • /queue — เช็กคิว OCR ที่กำลังประมวลผล
  • /failed · /retry_failed — ดู/รันงานที่ล้มเหลวใหม่
  • /sheetlink — ขอลิงก์ Google Sheets
  • /help · /start — เมนู/วิธีใช้งาน
